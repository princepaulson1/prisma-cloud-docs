1. Encryption providers are not appropriately configured

1.1. Policy Details

Prisma Cloud Policy ID

Severity

HIGH

Subtype

Build

Frameworks

Kubernetes,Helm,Kustomize

1.2. Description

Where etcd encryption is used, appropriate providers should be configured. Where etcd encryption is used, it is important to ensure that the appropriate set of encryption providers is used. Currently, the aescbc, kms and secretbox are likely to be appropriate options.
